# Hospital-Management
A Hospital Management System built using Spring Boot with Spring Data JPA and Hibernate, focused on database modeling, ORM mapping, and efficient data access. The project demonstrates real-world relational mappings, advanced JPA querying, pagination, projection, and performance optimization techniques using Hibernate.

Relationships:
One Patient → Many Appointments
One Doctor → Many Appointments
Many Doctors ↔ Many Departments
One Patient → One Insurance
